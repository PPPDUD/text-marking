# text-marking
An attempt at defeating particulary-lazy plagiarists by embedding invisible Unicode characters into text, using a system that I call TEPCS (**T**he **E**xperimental **P**lagiarism **C**atching **S**ystem).

## Methodology
### Marking
Texts with TEPCS applied to them contain repeated uses of a fingerprinting pattern, which contains a zero-width non-joiner followed by any positive, whole number of zero-width joiners, followed by another zero-width non-joiner; the amount of zero-width joiners employed is used as an identification mechanism to assist in tracing the source of a work. These are to be applied immediately before or after each space.

These patterns can be trivially viewed and/or removed with a sufficiently-advanced text editor or by manually retyping each letter by hand, but doing so is a waste of time better spent on producing original works, and the hope is that many plagiarists won't even notice the application of the TEPCS system until they have already published a plagiarized version.
